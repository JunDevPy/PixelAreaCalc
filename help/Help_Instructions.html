Гайд по использованию приложения для расчёта площади фигур на изображениях

1. Запуск приложения

Запустите EXE файл приложения на вашем компьютере. Откроется главное окно с графическим интерфейсом.

2. Загрузка изображений

  • Выберите папку с изображениями для пакетной обработки кнопкой Обработать всю папку или загрузите один файл через кнопку Обработать один файл.
  • Поддерживаемые форматы изображений: .png, .jpg, .jpeg, .tif, .tiff, .bmp.
  • Важно: на изображениях не должно быть рамок или границ — это может исказить результаты обработки, либо изображение не будет обработано вовсе!

3. Настройки обработки

  • Параметр Минимальная площадь (px) отвечает за фильтрацию мелких объектов и шумов.
  • Если приложение ошибочно распознаёт текст, артефакты или другие структуры как фигуры, увеличьте это значение. В качестве ориентира можно использовать, что примерно в 1 квадратном сантиметре — 10000–12000 пикселей.
  • Вы также можете настроить Разрешение (DPI) и Тип связности, влияющие на качество и точность распознавания.

4. Просмотр результатов

  • Результаты отображаются в таблице с информацией о файле, номере фигуры, количестве пикселей и площади.
  • При клике на строку таблицы показывается соответствующее изображение с выделенными фигурами.
  • Результаты сохраняются в папку results по умолчанию, которую можно изменить.

5. Режим отладки и логи

Для отладки приложения включён debug режим. Логи работы приложения могут быть полезны при возникновении ошибок или неточностей.

Файлы логов обычно находятся в папке запуска приложения или в папке с результатами results. Откройте их любым текстовым редактором для просмотра подробной информации.

Совет: Если столкнулись с некорректными результатами, попробуйте сначала увеличить Минимальная площадь (px) и проверить изображения без рамок.

6. Дополнительные функции

  • Кнопки увеличения/уменьшения масштаба просмотра изображений.
  • Открытие папки с результатами из интерфейса приложения.
  • Экспорт результатов в Excel для дальнейшего анализа.
  • Очистка таблицы результатов кнопкой Очистить результаты.
